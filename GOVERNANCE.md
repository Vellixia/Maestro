# Maestro Governance

## Maintainers

Maestro is currently maintained by the project's founding team. As the project grows, governance will evolve to include community maintainers.

## Decision Making

- Technical decisions are made by maintainers through consensus
- Architectural changes should be discussed via GitHub Issues before implementation
- Breaking changes require a maintainer review and approval

## Contributions

See [CONTRIBUTING.md](CONTRIBUTING.md).

## Roadmap

The project follows the build phases outlined in the [README](README.md#build-phases). Major feature priorities are managed through GitHub Issues with the appropriate labels.
